Stock Portfolio Tracker

This is a simple Python application that calculates the total investment value of a stock portfolio.

Features:
- User enters stock names and quantities
- Uses predefined stock prices
- Calculates total investment value
- Simple and user-friendly interface

Concepts Used:
- Dictionary
- User Input/Output
- Loops
- Conditional Statements
- Basic Arithmetic Operations

Technology Used:
- Python
